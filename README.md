# 👨‍💻 Portfólio Pessoal - Eliesdras Dias

## 🚀 Sobre o Projeto
Meu portfólio digital oficial, desenvolvido do zero para centralizar meus projetos, habilidades técnicas e informações de contato. 

A arquitetura da interface foi projetada com foco em **Responsividade (Mobile-First)** e **Experiência do Usuário (UX)**, garantindo uma navegação limpa, rápida e intuitiva em qualquer tamanho de tela (Desktop, Tablet ou Mobile).

🔗 **[Acesse o Portfólio em Produção aqui](https://eliesdrasdias.github.io/eliesdras-portfolio/)**

## 🛠️ Tecnologias e Arquitetura
Este projeto foi construído utilizando as seguintes tecnologias:
* **HTML5:** Estrutura semântica e acessível.
* **CSS3:** Estilização customizada, Flexbox e Media Queries para adaptação de telas.
* **JavaScript:** Lógica de manipulação de DOM (desenvolvimento do Menu Hambúrguer interativo para dispositivos móveis).
* **Git & GitHub:** Versionamento contínuo e *Deploy* via GitHub Pages.

## ✨ Funcionalidades Principais
* **Navegação Responsiva:** Menu dinâmico que se adapta automaticamente ao dispositivo do usuário.
* **Vitrine de Projetos:** Seção dedicada para exposição de código e aplicações em produção.
* **Acesso Direto:** Links otimizados para contato profissional (LinkedIn e E-mail).
